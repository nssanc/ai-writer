# 医学影像AI文献综述方法整合计划

## 概述

将系统化的7阶段文献综述写作方法整合到现有系统中，提升综述质量和写作效率。

## 整合内容

### 1. 数据库扩展

新增表：
- `writing_templates` - 存储写作模板
- `literature_categories` - 文献分类管理
- `quality_checklist` - 质量检查项

### 2. 新增功能模块

#### 2.1 写作模板系统
- 标准综述结构模板
- 段落写作模板
- 对比表格模板
- 图表占位符模板

#### 2.2 文献分类管理
- 按方法分类（CNN/Transformer/GAN等）
- 按应用分类（分割/检测/分类等）
- 按数据集分类
- 文献矩阵视图

#### 2.3 写作风格指南
- Hedging language 建议
- 引用规范检查
- 段落结构模板
- 常用学术短语库

#### 2.4 质量检查系统
- 结构完整性检查
- 引用覆盖率检查
- 语言风格检查
- 对比表格检查

### 3. 工作流程优化

#### 现有流程：
1. 上传参考文献
2. 风格分析
3. 生成写作指南
4. 生成撰写计划
5. 搜索文献
6. AI写作

#### 优化后流程（7阶段）：
1. **项目初始化** - 选择综述类型和模板
2. **文献收集** - 多源搜索（arXiv/PubMed/Zotero）
3. **文献分类** - 按方法/应用/数据集分类
4. **大纲开发** - 基于模板生成详细大纲
5. **分段写作** - 使用模板逐段写作
6. **表格图表** - 生成对比表格和图表
7. **质量检查** - 自动检查和改进建议
