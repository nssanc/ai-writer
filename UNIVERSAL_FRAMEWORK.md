# 通用文献综述框架系统设计

## 核心理念

创建一个**领域无关**的综述写作系统，通过关键词驱动的方式适应不同研究领域。

## 系统架构

### 1. 关键词管理系统

用户输入研究主题关键词，系统自动：
- 提取核心概念
- 推荐相关关键词
- 生成搜索策略
- 指导文献分类

### 2. 灵活的综述结构

提供多种标准模板，用户可选择或自定义：

**模板A：经典综述结构**
1. 引言（背景、研究问题、综述范围）
2. 方法（文献检索策略、纳入排除标准）
3. 主体内容（按主题/时间/方法分类）
4. 讨论（趋势、挑战、未来方向）
5. 结论

**模板B：系统综述结构**
1. 摘要
2. 引言
3. 方法学
4. 结果（文献筛选、质量评估、综合分析）
5. 讨论
6. 结论

**模板C：叙事综述结构**
1. 引言
2. 历史发展
3. 当前研究现状
4. 关键问题与挑战
5. 未来展望
6. 结论

### 3. 智能文献分类

基于关键词自动分类：
- 方法类（算法、技术、工具）
- 应用类（场景、案例、实践）
- 理论类（概念、模型、框架）
- 数据类（数据集、基准、评估）

### 4. 写作辅助功能

- 学术用语建议
- 段落结构模板
- 引用格式检查
- 逻辑连贯性检查

## 实施优先级

**第一阶段（立即实施）：**
1. 关键词管理界面
2. 基础模板系统
3. 文献标签功能

**第二阶段（后续）：**
1. AI辅助分类
2. 写作建议
3. 质量检查
